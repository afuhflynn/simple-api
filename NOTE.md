# Daily Learning Internship Program - Complete Web Dev Roadmap

## Notice

- Create a repository on GitHub: `<Learning_Js_Your_name>`.
- For each day, create a folder `day_X` (e.g., `day_1`) and work inside it.
- **Minimum 10 commits per day**.
- Share daily reports including repository folder URL.
- Projects should be deployed when possible.

# Week 0: Git & GitHub Basics

### Day 0: Introduction to Git

**Objectives:**

- Learn version control and why Git is important.
- Basic commands: `git init`, `git add`, `git commit`, `git status`.
- Connect repository to GitHub: `git remote add origin <url>`.

**Resources:**

- [W3Schools Git Intro](https://www.w3schools.com/git/git_intro.asp)
- [Simplilearn Git Commands](https://www.simplilearn.com/tutorials/git-tutorial/git-commands)

**Mini Project:** Initialize repo and commit a README.md.

**Report:** Share daily GitHub folder URL with 10+ commits.
